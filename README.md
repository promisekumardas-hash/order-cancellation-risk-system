# Order Cancellation Risk System

A machine learning powered web application to predict the risk of order cancellation
before delivery. Designed to help operations teams take early action and reduce losses.

## Features
- Single order risk prediction
- Batch CSV upload
- Risk score and risk level
- Simple Streamlit-based UI

## Tech Stack
- Python
- Scikit-learn
- Streamlit
- Pandas

## How to Run Locally

1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the app:
   streamlit run app.py

## Use Case
Helps e-commerce and logistics teams identify high-risk orders and proactively confirm deliveries.
